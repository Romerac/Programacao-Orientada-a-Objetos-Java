


# Exercício – Classe Retângulo em Java
![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)
## Descrição

Este exercício tem como objetivo a implementação de uma classe que representa um retângulo, utilizando a linguagem Java e os conceitos fundamentais da Programação Orientada a Objetos (POO). A atividade propõe a criação de atributos para armazenar as dimensões da figura, como altura e base, além da implementação de métodos responsáveis por manipular e acessar esses dados.

Inicialmente, são desenvolvidos métodos de acesso (getters), que permitem consultar os valores dos atributos de forma controlada, respeitando o princípio de encapsulamento. Em seguida, são implementados métodos responsáveis por realizar cálculos relacionados à figura geométrica, como área e perímetro.

Posteriormente, o exercício é expandido com a implementação de um método adicional para o cálculo da diagonal do retângulo, utilizando o Teorema de Pitágoras. Essa etapa reforça a aplicação de lógica matemática dentro do contexto da programação.

Dessa forma, a atividade contribui para o entendimento da organização de classes, manipulação de atributos por meio de métodos e desenvolvimento de funcionalidades baseadas em cálculos, consolidando conceitos essenciais da programação orientada a objetos.
