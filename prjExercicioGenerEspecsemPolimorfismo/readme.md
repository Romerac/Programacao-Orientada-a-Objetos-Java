
# Exercício – Generalização/Especialização sem polimorfismo
![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)


## Descrição

Este exercício tem como objetivo a implementação de um sistema em Java baseado nos conceitos de herança e polimorfismo, utilizando a Programação Orientada a Objetos (POO). A proposta parte da modelagem de uma classe genérica que representa uma pessoa, contendo atributos básicos como nome e telefone, a partir da qual são derivadas subclasses específicas.

As subclasses implementadas representam diferentes especializações da classe base, como um instrutor e um cliente, cada uma com seus próprios atributos e comportamentos adicionais. Essa estrutura permite explorar o conceito de herança, onde as subclasses reutilizam características da classe principal, além de introduzir novas funcionalidades específicas.

O exercício também aborda a utilização de construtores com parâmetros, garantindo a correta inicialização dos objetos no momento de sua criação, bem como o uso de métodos de acesso para manipulação dos dados.

Além disso, a atividade envolve a criação de uma aplicação responsável por instanciar objetos de cada uma das subclasses, possibilitando a interação com esses objetos e a verificação prática do comportamento herdado e especializado.

Dessa forma, o exercício contribui para a compreensão dos conceitos de generalização, especialização e reutilização de código, fundamentais para o desenvolvimento de sistemas orientados a objetos mais organizados e escaláveis.
