# Exercício – Classe Livros em Java
![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)


## Descrição

Este exercício tem como objetivo a implementação de um sistema simples de gerenciamento de livros utilizando a linguagem Java, com base nos princípios da Programação Orientada a Objetos (POO). A atividade propõe a modelagem de uma classe responsável por representar um livro, contemplando atributos como identificação, título, situação de disponibilidade e valor de multa por atraso.

A partir dessa modelagem, o sistema permite simular operações comuns em um contexto de empréstimo de livros, como consulta de informações, realização de empréstimos e devoluções. Além disso, incorpora regras de negócio, como a verificação da disponibilidade do livro e o cálculo de multa em casos de atraso na devolução. Tudo de forma extremamente limitada e simplificada apenas para fixação de conceitos.

A aplicação também envolve a interação com o usuário por meio de um menu no terminal, utilizando entrada de dados via teclado, o que possibilita testar o comportamento da classe em diferentes cenários.

Dessa forma, o exercício contribui para a compreensão prática de conceitos fundamentais da POO, como encapsulamento, controle de estado do objeto e implementação de métodos, promovendo a organização e reutilização de código.
